## 1) Problem Description
This project simulates an autonomous cooperative multi-robot system in Gazebo.
A mobile robot navigates to a pickup area and delivers an object to a handover zone.
Then a robotic manipulator detects the target (using tags/camera) and performs a pick-and-place task.

**Robots**
- Mobile robot: TurtleBot3 (differential drive)
- Manipulator: UR5 / Franka Panda (with gripper)
